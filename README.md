# Job Data Pipeline

This is a data engineering project that reads job data from a CSV file, cleans it using Python, and saves the cleaned output as a new CSV file.

## Project Overview

This project demonstrates a simple ETL-style workflow:

1. Extract job data from a CSV file
2. Transform and clean the data with Python and pandas
3. Load the cleaned data into a new CSV file for further analysis

## Tech Stack

- Python
- pandas
- SQL
- Git
- GitHub
- VS Code
